# React Webpack Skeleton

## Setup

- `git clone https://github.com/knledg/react-webpack-skeleton.git your-project`
- `cd your-project`
- `rm -rf .git`
- `git init && git remote add origin <sshUrl>`
- `npm install`
- `cp dev.env .env`
- `npm i -g foreman` (if you don't have it globally installed already)
- `make client`
